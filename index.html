METHOD 01 — BRACKETING — ROOT FINDING

Bisection
Method

طريقة التنصيف — تقسيم المجال لنصفين متساويين

THE IDEA

إذا كان f(a) و f(b) مختلفا الإشارة فيوجد جذر في المجال [a,b] — نقسم المجال لنصفين ونختار النصف الذي يحتوي على تغيير الإشارة.

ALGORITHM
  • تحقق: f(a)·f(b) < 0
  • احسب c = (a+b)/2
  • إذا f(a)·f(c) < 0 → b = c
  • إذا f(c)·f(b) < 0 → a = c
  • كرر حتى |error| < ε

METHOD 02 — BRACKETING — REGULA FALSI

False
Position

طريقة الموضع الكاذب — يصل أسرع لكن قد يبطئ إذا كانت الدالة منحنية

THE IDEA

نرسم خطاً بين النقطتين (a, f(a)) و (b, f(b)) ونجد تقاطعه مع محور x — أسرع من الثنائي لأنه يأخذ شكل الدالة بعين الاعتبار.

ALGORITHM
  • تحقق: f(a)·f(b) < 0
  • احسب c = b − f(b)·(b−a)/(f(b)−f(a))
  • إذا f(a)·f(c) < 0 → b = c
  • إذا f(c)·f(b) < 0 → a = c
  • كرر حتى |error| < ε

METHOD 03 — OPEN — NEWTON-RAPHSON

Newton's
Method

طريقة نيوتن رافسون — تقارب تربيعي سريع جداً

THE IDEA

نمد المماس للمنحنى عند نقطة xi — نقطة تقاطعه مع محور x تعطينا التقدير التالي xi+1. يتقارب تقارباً تربيعياً لكنه يحتاج مشتقة الدالة.

ALGORITHM
  • ابدأ بتخمين x₀
  • احسب x₁ = x₀ − f(x₀)/f'(x₀)
  • احسب الخطأ ε = |x₁−x₀|/|x₁| × 100
  • كرر حتى ε < المطلوب
  • المشتقة تُحسب عددياً تلقائياً

METHOD 04 — OPEN — FIXED POINT ITERATION

Fixed Point
Iteration

طريقة النقطة الثابتة — إعادة ترتيب f(x)=0 إلى x=g(x)

THE IDEA

نعيد كتابة f(x)=0 على شكل x=g(x) ثم نكرر xi+1 = g(xi). يتقارب إذا كان |g'(x)| < 1 عند الجذر.

ALGORITHM
  • أدخل g(x) بحيث x = g(x)
  • ابدأ بتخمين x₀
  • احسب xnew = g(x)
  • ε = |xnew−x|/|xnew| × 100
  • كرر حتى ε < المطلوب
INPUT PARAMETERS
Enter g(x) such that x = g(x). Example: for f(x)=-0.9x²+1.7x+2.5=0 → g(x)=sqrt(1.9x+2.8)

METHOD 05 — OPEN — SECANT METHOD

Secant
Method

طريقة القاطع — نيوتن بدون مشتقة

THE IDEA

نقرّب المشتقة بفارق محدود بين تخمينين متتاليين xi-1 و xi. لا يحتاج مشتقة صريحة ولكن يتقارب بشكل أبطأ قليلاً من نيوتن.

ALGORITHM
  • ابدأ بتخمينين x₋₁ و x₀
  • x₁ = x₀ − f(x₀)·(x₋₁−x₀)/(f(x₋₁)−f(x₀))
  • ε = |x₁−x₀|/|x₁| × 100
  • x₋₁ = x₀ ، x₀ = x₁
  • كرر حتى ε < المطلوب

Gaussian
Elimination

حذف غاوس — مع استبدال خلفي (مع الإزاحة الجزئية)

THE IDEA

ALGORITHM

  • إزاحة جزئية: بدّل الصف مع أكبر معامل
  • الحذف الأمامي: حوّل إلى مثلث علوي
  • الاستبدال الخلفي: احل من الأسفل للأعلى

METHOD 07 — DIRECT — LU DECOMPOSITION

LU
Factorization

تحليل LU — تحليل A=LU ثم حل نظامين مثلثيين

THE IDEA

نحلل A = L·U حيث L مثلث سفلي بواحدات قطرية وU مثلث علوي. ثم نحل Lc=b للحصول على c، ثم Ux=c للحصول على x.

ALGORITHM
  • احسب عوامل الحذف mᵢⱼ = aᵢⱼ/aⱼⱼ
  • U = المصفوفة المثلثية العلوية
  • L = مصفوفة عوامل الحذف (قطرها 1)
  • حل Lc=b ثم Ux=c

ALGORITHM

  • لكل xᵢ: أبدل العمود i بـ b
  • احسب det(Aᵢ)
  • xᵢ = det(Aᵢ) / det(A)

METHOD 10 — OPTIMIZATION — 1D SEARCH

Golden Section
Search

بحث القطاع الذهبي — إيجاد الحد الأقصى أو الأدنى للدوال أحادية البُعد

THE IDEA

نختار نقطتين داخليتين بنسبة الذهب φ=(√5-1)/2 ≈ 0.618 ونضيّق المجال بشكل تدريجي. كل تكرار يقلص المجال بمقدار 61.8%.

ALGORITHM
  • d = φ·(xu−xl)
  • x1 = xl + d ، x2 = xu − d
  • إذا f(x2) > f(x1): xu = x1
  • إذا f(x1) > f(x2): xl = x2
  • كرر حتى |xu−xl| < ε

METHOD 11 — OPTIMIZATION — GRADIENT METHOD

Steepest
Ascent/Descent

طريقة أشد الانحدارات — للدوال ثنائية الأبعاد f(x,y)

THE IDEA

نحسب التدرج ∇f=(∂f/∂x, ∂f/∂y) عند النقطة الحالية ونتحرك في اتجاهه. في كل خطوة نجد أمثل خطوة h على المسار الجديد.

ALGORITHM
  • احسب ∂f/∂x و ∂f/∂y عند (x,y)
  • اتجاه الصعود: d = (∂f/∂x, ∂f/∂y)
  • g(h) = f(x+h·dx, y+h·dy)
  • أوجد h* يعظّم/يصغّر g(h)
  • حدّث: x = x+h*·dx, y = y+h*·dy
